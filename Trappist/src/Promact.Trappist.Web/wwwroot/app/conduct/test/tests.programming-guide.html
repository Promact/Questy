Programming Instructions

  1. Use Standard Input to take input and Standard Output to print output.
    • For C use: scanf for Inputs and printf for Outputs.
    • For C++ use: cin for Inputs and cout for Outputs.
    • For Java you can use BufferedReader or Scanner class for Inputs and System class for Output.
  2. GCC compiler is used to compile C and C++ code. Use header files accordingly.
  3. You should return zero (0) from main method in C and C++ language.
  4. Class name for Java should be strictly named as Program.
  5. Do not define your own Package Name in Object Oriented Programming language.
  6. Using third party library or making network calls through your code must be avoided.
Common Mistakes
  1. Do not print anything other than things instructed in the question. Printing user messages like Enter a number:,Enter value of x:, etc. (until specified in the question) will result in wrong answer.
  2. When printing result make sure you don't accidently print extra spaces however trailing newlines will be removed when evaluating.
